**BugSmasher-ApZz v3.0**

A visually spectacular neon cyberpunk bug-squashing arena game built with Vite + React + TypeScript + Canvas 2D.

## Features
- Real-time hitstop, screen shake, combo multipliers
- Modular visual juice systems
- Wave-based gameplay with escalating difficulty
- Faithful recreation of the 2026 Visual Overhaul design

## Quick Start
```bash
npm install
npm run dev
```

## Vision
Enterprise-grade web game with future offline voice AI integration.
